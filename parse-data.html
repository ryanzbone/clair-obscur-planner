Clair Obscur Data Parser

This tool will parse the Picto data from the data.html file and convert it to a JSON format that can be used in the Character Planner application.

Steps:

  1. Make sure data.html is in the same directory as this file
  2. Click the "Load and Parse Data" button below
  3. Review the parsed data
  4. Click "Save to JSON" to download the data as a JSON file
